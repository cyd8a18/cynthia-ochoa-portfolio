# Cynthia Ochoa - Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, skills, education, and achievements as a Robotics and AI Engineer.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Multiple Sections**:
  - Hero Section with call-to-action buttons
  - About Me with key statistics
  - Work Experience timeline
  - Technical Skills organized by category
  - Languages and proficiency levels
  - Education history
  - Achievements and awards
  - Contact information
- **Smooth Interactions**: Scroll animations, hover effects, and interactive elements
- **SEO Optimized**: Proper semantic HTML structure
- **Fast Loading**: Lightweight CSS and JavaScript

## 🚀 Quick Start

1. **Clone or download this repository**
2. **Open `index.html` in your web browser** to view the website locally

## 🌐 Deploy to GitHub Pages

### Method 1: Using GitHub Pages (Recommended)

1. Create a new repository on GitHub named `<username>.github.io`
2. Clone the repository locally:
   ```bash
   git clone https://github.com/<username>/<username>.github.io.git
   ```
3. Copy all files from this portfolio folder to the repository folder
4. Push to GitHub:
   ```bash
   git add .
   git commit -m "Initial portfolio commit"
   git push origin main
   ```
5. Your website will be live at `https://<username>.github.io`

### Method 2: Using Project Pages

1. Push this code to a repository named `portfolio`
2. Go to repository settings → Pages
3. Select `main` branch as the source
4. Your website will be live at `https://<username>.github.io/portfolio`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6f1e51;      /* Main color */
    --secondary-color: #d4a373;    /* Accent color */
    --accent-color: #f4e4c1;       /* Light accent */
    --dark-bg: #1a1a1a;
    --light-bg: #f8f9fa;
}
```

### Content
Update the following in `index.html`:
- Navigation links
- Hero section text
- About section content
- Experience timeline entries
- Skills and categories
- Education entries
- Contact information and social links

### LinkedIn & GitHub Links
Update the social links in the contact section:
```html
<a href="https://linkedin.com/in/your-username" target="_blank">
<a href="https://github.com/your-username" target="_blank">
```
